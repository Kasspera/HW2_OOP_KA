# HW2_OOP_KA

Проект для работы с рецептами и списком покупок

В проекте есть классы Ingredient, Recipe, ShoppingList и DietaryRecipe.

## Установка

```bash
git clone https://github.com/Kasspera/HW2_OOP_KA.git
cd HW2_OOP_KA
pip install -r requirements.txt
```

## Использование

Код находится в файле `main.py`.

Чтобы запустить тесты:

```bash
pytest
```
